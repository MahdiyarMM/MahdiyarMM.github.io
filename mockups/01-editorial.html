§ 01 — Featured Work

In Long Form.

Three deep dives that shaped the rest of the work — a debiasing framework for vision-language models, a federated approach to domain generalization, and an open dataset that quietly became a benchmark.

Fig. 1 — PRISM disentangles biased clusters in CLIP-style embeddings.
ICCV 2025  ·  Vision-Language

PRISM: a data-free way to debias what the model already learned.

Most debiasing pipelines need labeled bias attributes, paired data, or a curated counterfactual set. PRISM throws all of that out and asks a different question — can we untangle the biased axes inside an existing CLIP encoder using only language?

The method drives a small linear correction over frozen embeddings using bias prompts generated by an LLM. No retraining, no labels. Across nine benchmarks it matches or beats supervised baselines, and it generalizes to tasks the original CLIP model was never aligned to.
